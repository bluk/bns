Global Variables

The following global variables are available globally.

  • BNSDefaultCallbackQueue

    The default callback queue which is used when an event occurs which requires a callback and the connection, stream, or listener is not started. Most commonly, this is used for logging events and transitioning to the cancelled state if an instance was not started with a queue. It must be a serial queue. You should only set this queue when there are no BNS objects which may issue a callback (e.g. at the beginning of your program).
